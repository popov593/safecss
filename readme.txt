== Description ==

This is a WooCommerce plugin, that allows you adding custom CSS to your website, without being scared changing the theme or upgrading will delete the progress. Find me under Appearance > Safe CSS.

== Installation and use ==

1. Upload the entire 'safecss' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go Appearance > Safe CSS.
